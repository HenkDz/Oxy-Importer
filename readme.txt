== Description ==

### Zoro Lite: an accompanying plugin to access the design set that is managed by the [Asura](https://thelostasura.com) plugin.

You should install the Zoro Lite plugin on your own WordPress website to access the design set from your seller.

#### Access Designset in Oxygen Editor

You can access the design sets collection right in your Oxygen Editor.

### Further Reading

For more info on selling designs built with Oxygen Builder, check out the following:

* The [Zoro Lite](https://thelostasura.com/zoro-lite) official homepage.

== Installation ==

Please [read documentation](https://thelostasura.com/docs/zoro) on Zoro Lite website for instruction how to install Zoro Lite plugin. 

== Frequently Asked Questions ==

= Who does need Zoro Lite plugin? =

If you are purchasing Oxygen Builder design sets that managed by Asura plugin, 
You should use the Zoro Lite plugin on your own WordPress website to access the 
design set from your seller. Please ask your design set seller for complete information.
